# Gabriel Oristides | Portfólio

Portfólio pessoal moderno, responsivo e animado, criado com HTML, CSS e JavaScript puro.

## Preview

Abra o arquivo `index.html` no navegador ou publique pelo GitHub Pages.

## Estrutura

- `index.html`: conteúdo principal da página
- `styles.css`: layout, responsividade e animações visuais
- `script.js`: partículas, interações, terminal animado e efeitos de scroll
- `assets/`: imagens do portfólio
